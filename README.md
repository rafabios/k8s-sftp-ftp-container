# k8s-sftp-ftp-container


### This Dockerfile contains emberstack/sftp and a vsftpd script to create an FTP single user.


## Variables FTP

```FTP_ROOT = "DESTNATION FOLDER" ``` 
```FTP_USER = "FTP USER NAME" ``` 
```FTP_PASS = "FTP PASSWORD" ```
